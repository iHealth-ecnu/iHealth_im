## iHealth_im
iHealth 项目的即时通讯后台

### 使用技术
* nodejs
* socket.io

### 使用说明
1. 安装 nodejs 环境  
装好之后运行以下代码测试 nodejs 是否生效
```
node -v
```

2. 安装 socket.io

3. 启动 iHealth_im 后台程序
```
sh iHealth_im.sh start
```
